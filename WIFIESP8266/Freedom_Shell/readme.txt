This is an example project for the Freescale Freedom board (Kinetis KL25Z Freedom Board).
Features:
- FreeRTOS with statistics enabled
- 3 RGB LED components with PWM and shell interface
- Two configurations: one with SD card (Arduino Data Logger) support and one without.
- Shell component which can use both the virtual USB CDC interface (OpenSDA) and the native K25Z USB port (USB CDC)

More information: http://mcuoneclipse.com/
